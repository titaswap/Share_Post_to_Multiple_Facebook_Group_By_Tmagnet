# Share Post to Multiple Facebook™ Groups | Safe Version

This is a strict specific-purpose browser extension designed to automate sharing posts to Facebook groups **safely** and **securely**.

## 🛡️ Security Certified
This extension has been audited and refactored to ensure:
- **No Background Scripts:** Zero persistent processes.
- **No External Network Calls:** The network is physically disabled via a strict `kill-switch`.
- **Local Storage Only:** No data is ever synced to the cloud.

## 🚀 Features
- **One-Click Scanning:** Finds all your groups instantly.
- **Human-Like Delays:** Random wait times to avoid spam detection.
- **Offline Only:** Runs entirely on your machine.

## 📦 Installation (Developer Mode)
1. Clone this repo.
2. Go to `chrome://extensions`.
3. Enable "Developer Mode".
4. Click "Load Unpacked" and select this folder.

## ⚠️ Disclaimer
This tool automates user actions. Use responsibly to avoid Facebook restrictions. Recommended sharing limit: 20-30 groups per day.
